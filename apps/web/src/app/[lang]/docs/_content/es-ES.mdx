# Plotwist

**Plotwist** nació para unir todo en una sola plataforma. A diferencia de otras opciones como Letterboxd, que se centra solo en películas, aquí puedes gestionar y evaluar películas, series, animes, doramas y mucho más.

Nuestro objetivo es simple: ofrecer un espacio completo y elegante para que organices tus listas, sigas lo que has visto y lo que quieres ver, todo en un solo lugar.

## Recursos

- **Evaluaciones**  
  Lee y comparte opiniones sobre películas, series, animes y doramas.

- **Gestión de Listas**  
  Organiza tus listas para mantener el control de lo que has visto y lo que aún quieres ver.

- **Watchlist**  
  Una lista práctica para seguir tus futuras maratones.

- **Archivo TMDB**  
  Integración con TMDB para acceder a un amplio catálogo de contenidos con información completa.

## Futuro

Planeamos expandirnos para incluir **comunidades**, permitiendo que los usuarios interactúen y compartan aún más sus pasiones y descubrimientos. Nuestra visión es hacer de **plotwist** el punto de encuentro definitivo para todos los aficionados al entretenimiento audiovisual.
